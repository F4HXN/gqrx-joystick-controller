# GQRX Joystick Controller

Contrôleur joystick pour GQRX permettant de piloter facilement la radio logicielle GQRX.


## Utilisation

1. Démarrer GQRX
2. Activer le contrôle à distance
3. Brancher le joystick
4. Lancer le programme
